Здесь будет ваш запрос

Сборка основы запроса

Введите основу запроса: регистр шрифта и знаки пунктуации не важны. Для поиска точных совпадений используйте кавычки, для замены слова или фразы используйте "*", для поиска по хештегу "#".

Настройка типа данных

Укажите тип файлов, которые будут в результатах выдачи. Если тип данных не будет указан, будут искаться страницы по умолчанию. В выдаче документы разных типов будут перемешаны.

Исключения

Исключения указываеются следующим образом: " -исключение1 -(исключение 2)". Если в результах поиска будут встречены исключенные слова, они не отобразятся на странице выдачи. Можно указать несколько исключений.

Поиск на конкретном сайте

Для поиска на конкретном сайте нужно указать его адрес. Адрес указывается следующим образом: "htmlbook.ru". По умолчанию поиск осуществляется по всем сайтам. Указать можно только один адрес.

Настройка даты

Для уточнения результатов поиска можно указать дату. Можно указать как интервал из двух чисел, так и одно число. Формат: от 2010 до 2022. И в левом, и в правом поле может быть только одно число.
